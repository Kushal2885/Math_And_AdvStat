# Project Overview

This project analyzes student academic performance using statistical methods, probability concepts, data visualization techniques, and basic linear algebra operations in Python.

The dataset contains information about students including their age, subject scores, study hours, and pass/fail status. The objective is to extract meaningful insights and apply theoretical concepts learned in Statistics, Data Science, and Python Programming.

---

# Dataset Description

The dataset consists of 5000 student records with the following attributes:

| Column Name   | Description                |
| ------------- | -------------------------- |
| Student_ID    | Unique Student Identifier  |
| Age           | Student Age                |
| Math_Score    | Mathematics Score          |
| Science_Score | Science Score              |
| English_Score | English Score              |
| Hours_Studied | Daily Study Hours          |
| Pass_Fail     | Pass (1) / Fail (0) Status |

# Key Findings

* Students studying more than 5 hours have a significantly higher probability of passing.
* Math scores approximately follow a normal distribution.
* Science scores show moderate variation among students.
* Strong positive relationship exists between Math and Science performance.
* Increased study hours generally lead to improved academic performance.
