# Derivable Judgement – Inferential Statistics Project

## Overview

This project demonstrates the implementation of inferential statistics concepts using Python and a health dataset. The analysis is performed in a Jupyter Notebook using statistical testing techniques.

---

## Dataset Used

* health_dataset.csv

The dataset contains health-related information such as:

* Age
* Weight
* BMI
* Smoking Status
* Diabetes
* Exercise Frequency
* Blood Pressure
* Cholesterol Level
* Glucose Level

---

## Tasks Performed

### 1. Hypothesis Formulation

Two hypotheses were created based on the dataset:

* Smoking vs Diabetes
* Exercise Frequency vs BMI

---

### 2. Confidence Interval Calculation

Confidence intervals were calculated for:

* Age
* Weight

---

### 3. Critical Value and p-value

* Critical value was calculated using statistical methods.
* p-value was used to interpret hypothesis testing results.

---

### 4. T-Test

A t-test was performed to compare:

* BMI of Smokers
* BMI of Non-Smokers

---

### 5. Chi-Square Test

A Chi-Square test was conducted to check the association between:

* Smoking Status
* Diabetes

---

### 6. ANOVA Test

ANOVA test was performed to analyze BMI differences across multiple age groups.

---

### 7. Covariance and Correlation

Covariance and correlation were calculated between:

* Age
* BMI

---

## Technologies Used

* Python
* Jupyter Notebook
* Pandas

* SciPy

---

## Conclusion

The project successfully demonstrated the use of inferential statistical techniques on a healthcare dataset using Python.
